# Estudos_NotebookLM
Estudos de engenharia de prompt

📚 Contexto e Objetivos

Este repositório faz parte de um desafio de projeto de um bootcamp voltado para Cybersegurança. O objetivo é organizar um caderno de estudos temático, reunindo anotações, conceitos, ferramentas e aprendizados adquiridos ao longo da minha jornada de aprendizado na área.

O assunto escolhido para este material é Cybersegurança, uma área fundamental da tecnologia da informação que tem como foco a proteção de sistemas, redes e dados contra ataques, acessos não autorizados e outras ameaças digitais.

🎯 Objetivos de Estudo

Com este repositório, busco:

Estudar cybersegurança de forma leve e progressiva, consolidando os principais conceitos da área.

Registrar anotações e aprendizados importantes ao longo do bootcamp e de estudos complementares.

Explorar diferentes caminhos profissionais dentro da cybersegurança, como análise de segurança, pentest, segurança defensiva, entre outros.

Criar um material de consulta pessoal, que possa ser revisado e expandido conforme avanço nos estudos.
